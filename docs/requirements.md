# 需求文档 — 婴儿喂养记录与统计分析 App

## 项目背景

新生婴儿的日常喂养、睡眠、排泄和营养补充是育儿过程中最核心的几项指标。传统的手写记录或手机备忘录存在信息分散、难以统计、无法多人协作等问题。本 App 为家庭用户提供一个集中式的记录与统计工具，支持多人共同记录同一个宝宝的数据，并通过可视化图表帮助家长了解宝宝的喂养趋势。

## 业务规则

1. 一个家庭 = 一个宝宝 + 多个用户
2. 邀请码 6 位字母数字，排除 0/O/1/I/l
3. 所有记录按 family_id 隔离，用户只能看到自己家庭的数据
4. 编辑/删除记录时需验证记录归属（同一 family）
5. 同一家庭同一时间只能有一个 running 状态的睡眠计时器
6. 月龄 = 当前日期 - 出生日期，精确到天（如"6个月11天"）
